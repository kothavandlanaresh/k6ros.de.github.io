Element Matrices in FEM

Introduction

Element matrices are used to represent the behavior of individual elements in a finite element model. They include the stiffness matrix, mass matrix, and damping matrix.

Stiffness Matrix

The stiffness matrix describes the relationship between forces and displacements. For a 1D element, it is:

        K = (E * A / L) * [1 -1
                           -1  1]

Mass Matrix

The mass matrix represents the distribution of mass in the finite element model. For a 1D element, it is often assumed to be:

        M = (ρ * A * L / 6) * [2 1
                              1 2]

Where ρ is the density of the material.

Damping Matrix

The damping matrix accounts for energy dissipation in the system. A common form for a 1D element is:

        C = α * K + β * M

Where α and β are damping coefficients.

Summary

Element matrices are fundamental components in FEM that capture the properties and behavior of individual elements, which are then assembled into the global system.
