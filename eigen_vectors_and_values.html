Understanding Eigenvalues and Eigenvectors

Introduction

Eigenvalues and eigenvectors are concepts in linear algebra that help us understand how transformations affect vectors. Here's a clear explanation using a 3x3 matrix.

Example Matrix

        A = [ 2 1 0
              1 2 0
              0 0 1 ]

1. Finding Eigenvalues

To find the eigenvalues, follow these steps:

  1. Subtract λ from the diagonal elements of the matrix:
    2.             A - λI = [ 2-λ 1   0
                       1   2-λ 0
                       0   0   1-λ ]
  2. Find the determinant of this matrix and set it equal to zero:
    3.             det(A - λI) = det([ 2-λ 1   0
                                 1   2-λ 0
                                 0   0   1-λ ])
                      = det([ 2-λ 1
                              1   2-λ ]) * (1-λ)
                      = [(2-λ)(2-λ) - 1] * (1-λ)
                      = [(λ-1)(λ-3)^2]

    Setting this to zero gives the eigenvalues:

                λ = 1
            λ = 3 (with multiplicity 2)

2. Finding Eigenvectors

For each eigenvalue, solve (A - λI) v = 0 to find the eigenvectors.

For λ = 1:

  1. Substitute λ = 1:
    2.             A - I = [ 1 1 0
                      1 1 0
                      0 0 0 ]
  2. Solve:
    3.             [ 1 1 0
              1 1 0
              0 0 0 ] [ x
                       y
                       z ] = [ 0
                               0
                               0 ]

    This simplifies to:

                    x + y = 0

    An eigenvector for λ = 1 is:

                    v = [ 1
                      -1
                       0 ]

For λ = 3:

  1. Substitute λ = 3:
    2.             A - 3I = [ -1 1 0
                       1 -1 0
                       0 0 -2 ]
  2. Solve:
    3.             [ -1 1 0
              1 -1 0
              0 0 -2 ] [ x
                        y
                        z ] = [ 0
                                0
                                0 ]

    This simplifies to:

                    x = y
                z = 0

    An eigenvector for λ = 3 is:

                    v = [ 1
                      1
                      0 ]

Summary

Eigenvalues describe how much stretching or squishing occurs, while eigenvectors describe the directions in which this stretching or squishing happens.
