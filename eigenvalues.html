Understanding Eigenvalues and Eigenvectors

Introduction

Eigenvalues and eigenvectors are fundamental concepts in linear algebra used to analyze linear transformations. In FEM, they are crucial for understanding how systems respond to different types of loads and constraints.

Example Matrix

        A = [ 2 1 0
              1 2 0
              0 0 1 ]

Finding Eigenvalues

To find the eigenvalues of matrix A, we solve the characteristic equation:

        det(A - λI) = 0

For matrix A, this is:

        det([ 2-λ 1   0
              1   2-λ 0
              0   0   1-λ ]) = (λ-1)(λ-3)^2

The eigenvalues are:

        λ = 1
        λ = 3 (with multiplicity 2)

Finding Eigenvectors

For each eigenvalue, solve the equation (A - λI)v = 0:

For λ = 1:

        A - I = [ 1 1 0
                  1 1 0
                  0 0 0 ]

Solving this yields:

        v = [ 1
              -1
               0 ]

For λ = 3:

        A - 3I = [ -1 1 0
                   1 -1 0
                   0 0 -2 ]

Solving this yields:

        v = [ 1
              1
              0 ]

Summary

Eigenvalues provide insight into the scaling factor of the transformation, while eigenvectors indicate the direction of this scaling.
